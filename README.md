# Windows
---
1. Установить **JDK**
2. Добавить переменные окружения

| Переменная | Описание |
|---|---|
|**JAVA_HOME**|C:\Program Files (x86)\Amazon Corretto\jdk21.0.10_9|
|**Path**|C:\Program Files (x86)\Amazon Corretto\jdk21.0.10_9\bin|
3. Проверка установленного JDK

| Команда | 
|---|
|**echo $JAVA_HOME**|
|**where java**|
4. Проверка версии JAVA

| Команда | 
|---|
|**$ java -version**|
5. Запуск Java-файлов на компьютере

| Команда | Описание |
|---|---|
|**javac -encoding UTF-8 <имя java-файла> / javac -encoding UTF-8 HelloJdk.java**|Скомпилировать исходный код в байт-код (проверить, что имя класса совпадает с именем файла)|
|**java <имя class-файла без расширения> / java -Dfile.encoding=UTF-8 HelloJdk**|Запустить файл с байт-кодом|
|---|---|
|**javac -d <имя папки> .java / javac -d bin -encoding UTF-8 .java**|Скомпилировать исходный код в байт-код для нескольких файлов. Запускается из папки src, сгенерированные файла складываются в папку bin (или другую указанную на этом месте)|
|**java -cp <имя папки, в которой лежат class-файлы> / java -cp bin Practicum**|Запустить файл с байт-кодом|

6. Создание .jar файла
Архив содержащий **файл манифеста META-INF/MANIFEST.MF** - специальный текстовый файл с описанием структуры архива

| Команда | Описание |
|---|---|
|**javac -d <имя папки> .java / javac -d bin -encoding UTF-8 .java**|Скомпилировать исходный код в байт-код для нескольких файлов|
|**jar cfe <имя jar-файла> <имя стартового класса> <список файлов> / jar cfe library.jar Practicum -C bin .**|Создать .jar файл|
|**java -jar <имя jar-файла> / java -jar library.jar**|Запустить .jar файл|
